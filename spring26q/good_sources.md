# Quantum Playgrounds

This repository is for people interested in playing around with quantum tools and frameworks. It contains a wide list of repositories and sources that might be helpful to get started with quantum systems, tools and algorithms.

## SDKs

- **Qiskit** (opensource, maintained by IBM)
  - Github Repo: [https://github.com/Qiskit/qiskit](https://github.com/Qiskit/qiskit)
  - Documentation: [https://quantum.cloud.ibm.com/docs/](https://quantum.cloud.ibm.com/docs/)
- **Munich Quantum Toolkit** (opensource, maintained by multiple European orgs)
  - Github Org: [https://github.com/munich-quantum-toolkit](https://github.com/munich-quantum-toolkit)
  - Paper: [The MQT Handbook : A Summary of Design Automation Tools and Software for Quantum Computing](https://ieeexplore.ieee.org/document/10646543)
  - Documentation: [https://mqt.readthedocs.io](https://mqt.readthedocs.io/)

## Simulators

- **Qiskit IBM Runtime Fake Provider**
  - API reference: [Fake Provider](https://quantum.cloud.ibm.com/docs/en/api/qiskit-ibm-runtime/fake-provider) (Contains multiple backends to try out)
- **Qiskit Aer** (Provided by IBM)
  - GIthub: [https://github.com/Qiskit/qiskit-aer](https://github.com/Qiskit/qiskit-aer)
- **Quirk: Quantum Circuit Simulator**
  - Online version: [https://algassert.com/quirk](https://algassert.com/quirk)
  - Github Repo (for self-hosting): [https://github.com/Strilanc/Quirk](https://github.com/Strilanc/Quirk)

## Cloud Environments

- **IBM Quantum Platform** (managed by IBM)
  - Qiskit Runtime: [Docs](https://quantum.cloud.ibm.com/docs/en/api/qiskit-ibm-runtime/runtime-service), [Github](https://github.com/Qiskit/qiskit-ibm-runtime)
  - Qiskit Functions: [https://quantum.cloud.ibm.com/docs/en/guides/functions](https://quantum.cloud.ibm.com/docs/en/guides/functions) (experimental, only premium)

## Hardware Tools

- **QubiC: Open source FPGA-based control and measurement system**
  - Paper: [QubiC: An open source FPGA-based control and measurement system for superconducting quantum information processors](https://arxiv.org/abs/2101.00071)
  - New paper: [QubiC 2.0: An Extensible Open-Source Qubit Control System Capable of Mid-Circuit Measurement and Feed-Forward](http://arxiv.org/abs/2309.10333)
  - Docs: [https://lbl-qubic.gitlab.io](https://lbl-qubic.gitlab.io/)
